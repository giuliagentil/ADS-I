Resumo do Semestre: Algoritmos e Lógica de Programação , Engenharia de Software e Projeto Integrador 

Engenharia de Software

Durante o primeiro semestre, a disciplina de Engenharia de Software proporcionou o estudo dos principais conceitos relacionados ao desenvolvimento de sistemas. A partir do estudo de caso do Sistema Web para Gestão de Pesquisas por Questionários, foi possível compreender as etapas de análise, planejamento e modelagem de software.

Entre os principais conteúdos estudados destacam-se:

* Levantamento de requisitos funcionais e não funcionais;
* Identificação dos atores do sistema;
* Elaboração de User Stories;
* Construção de casos de uso;
* Definição de fluxos principais e alternativos;
* Conceitos de autenticação e autorização;
* Princípios de segurança da informação;
* Planejamento e documentação de software.

Também foram definidos os requisitos do sistema, incluindo funcionalidades como cadastro de pesquisas, geração de senhas anônimas, aplicação de questionários e geração de relatórios estatísticos. A disciplina permitiu compreender a importância da documentação como base para o desenvolvimento de sistemas de qualidade.

Projeto Integrador

O Projeto Integrador teve como foco o desenvolvimento da proposta de um Sistema Web para Gestão de Pesquisas por Questionários, criado para oferecer mais segurança, anonimato e confiabilidade na aplicação de pesquisas institucionais.

Durante o semestre foram realizadas as seguintes atividades:

* Pesquisa e análise de plataformas já existentes;
* Estudo comparativo entre Google Forms, SurveyMonkey, Typeform e KoBoToolbox;
* Elaboração do Documento de Visão e Escopo;
* Desenvolvimento do Backlog Inicial;
* Definição dos requisitos do sistema;
* Criação do Pitch de apresentação do projeto;
* Definição dos perfis de usuários;
* Planejamento do MVP (Produto Mínimo Viável);
* Definição das tecnologias utilizadas no desenvolvimento.

Além disso, foram definidos os diferenciais da solução proposta, como a utilização de senhas únicas para garantir o anonimato dos participantes e impedir respostas duplicadas, aumentando a confiabilidade dos dados coletados.

Algoritmos e Lógica de Programação

Na disciplina de Algoritmos e Lógica de Programação foram estudados os fundamentos necessários para a construção de programas computacionais e resolução de problemas por meio da lógica.

Entre os conteúdos trabalhados destacam-se:

* Variáveis e tipos de dados;
* Estruturas condicionais;
* Estruturas de repetição;
* Vetores;
* Manipulação de caracteres;
* Fluxogramas;
* Desenvolvimento de algoritmos.

Como atividade prática, foi desenvolvido um algoritmo gerador de senhas utilizando o VisualG. O programa foi projetado para gerar senhas automáticas destinadas a diferentes categorias de usuários, armazenando os resultados em vetores e exibindo as informações ao final da execução.

O projeto possibilitou aplicar na prática os conceitos estudados, fortalecendo o raciocínio lógico e a compreensão das etapas envolvidas no desenvolvimento de soluções computacionais.
